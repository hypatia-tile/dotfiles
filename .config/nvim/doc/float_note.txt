*float_note.txt* Stateless persistent floating note (floating window) *float_note*
USAGE
  :FloatNote
  :FloatNote {width} {height}
      Open/close a centered floating scratch note.
      {width}/{height} can be ratios (0-1) or absolute cells (>=1).

MAPPING
  <leader>o    Toggle the floating note (see plugin/float_note.lua)

RATIONALE                                                             *float_note-rationale*
  Buffer vs Window:
    A buffer stores text; a window displays a buffer.
    We name the buffer "float://note" and look it up each time,
    so no Lua module state is needed.

  Persistence:
    The buffer uses |'bufhidden'|=hide, so closing the last window
    does not wipe the buffer-edits persist across toggles.

API                                                                   *float_note-api*
  opts.width   number (0-1 ratio or >=1 cells) default 0.8
  opts.height  number (0-1 ratio or >=1 cells) default 0.8
  opts.border  string one of "rounded"(default), "single", "double", "none", â€¦
  opts.title   string window title
